# 作業メモ

プログラミング言語 LuneScript の日本語のリファレンスサイト(HTML5)を作成します

- リファレンスサイトは artifacts 以下に作成してください
- 日本語は artifacts/ja, 英語は artifacts/en に作成してください
- LuneScript の定義は all.ja.org を参照してください
- all.ja.org の内用をまとめて再構成し、新しくリファレンスサイトを作成します
- 想定読者は、なんらかのプログラミング言語を習得したソフトウェアエンジニアです
- LuneScript を魅力的をソフトウェアエンジニアに伝わるように構成してください
- リファレンスサイトの構成は、 https://teal-language.org/book/index.html を参考にしてください
- 言語の機能をカテゴリ別に分けて再構成してください
- 各章は 5000 文字程度の内用にまとめてください
- サンプルを多めにいれてください
- 文法で不明確なことがあれば聞いてください
- 作業で一時的に必要なファイルは temp 以下に作成してください
- temp, artifacts 以外のファイルは変更しないでください


# サンプルの確認方法

- リファレンスに載せるサンプルは動作を確認してください。
- 動作を確認する場合、次の方法で確認できます。

```
$ cd test
$ ./lnsc sample.lns exe
```

- ここで sample.lns は、サンプルコードを保存したファイルです
- サンプルが正常な場合、エラーなく実行できます。

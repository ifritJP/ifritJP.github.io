- セキュリティを考慮したテキストの暗号化、復号ツールを作成した
- 実際のソースコードは https://github.com/ifritJP/cryptool_webext/tree/master を参照すること
- どこで利用できることと、安心して使えることを考慮して firefox の web extension として作成した
- Web extension として作成したことのメリット
  - PC/スマホで利用できる
  - ユーザが意図しない通信の規制を保証できる
- ただし、 web extension はブラウザによるキャッシュ、OS によるキャッシュが働く可能性がある
- これに対応するため、出来るだけキャッシュされないように対応している
- とはいえ、限界はあるので、その辺りのリスクはユーザ責任で運用が必要
- コードは AI とレビューしながら出来るだけセキュアなコードになるように開発
- コーディング自体は AI エージェントが行なっている
